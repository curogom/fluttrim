# ROADMAP

Fluttrim의 공개 로드맵입니다. 우선순위와 일정은 변경될 수 있습니다.

## v1 핵심 범위

1. Core 스캔/플랜/적용 안전 규칙 고도화
2. CLI 기능 완성도 개선(출력/진단/자동화 친화성)
3. Desktop UX 개선(프로젝트 관리/플랜 미리보기/적용 흐름)
4. macOS DerivedData 귀속 정확도 개선
5. Global cache 가시성/가드레일 강화
6. FVM 사용 현황 분석 및 정리 UX 정교화
7. 실행 이력(History) 저장/비교 개선
8. 문서/테스트/CI 품질 지속 강화

## 운영 관련 계획

- pub.dev: `fluttrim_core` 우선 공개
- Cloudflare Pages: 홍보 웹 지속 업데이트
- 오픈소스 문서/정책 세트 지속 보강
