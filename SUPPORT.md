# SUPPORT

Fluttrim 오픈소스 프로젝트 지원 정책입니다.

## 문의 채널

- 버그 제보: GitHub Issues (Bug Report 템플릿 사용)
- 기능 제안: GitHub Issues (Feature Request 템플릿 사용)
- 사용 문의: GitHub Issues에 `question` 성격으로 등록
- 보안 이슈: 공개 이슈 대신 `SECURITY.md` 절차 사용

## 이슈 작성 기준

- 재현 절차, 기대 동작, 실제 동작을 분리해 작성
- OS/Flutter/Dart 버전 정보를 포함
- 관련 로그(JSON/CLI 출력) 첨부 권장

## 응답 기준

- 커뮤니티 기반 운영으로 즉시 응답을 보장하지 않습니다.
- 우선순위는 재현 가능성, 영향도, 프로젝트 로드맵 기준으로 판단합니다.

## 언어

- 한국어/영어 이슈 모두 허용합니다.
