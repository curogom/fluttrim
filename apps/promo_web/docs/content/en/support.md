# Support Policy

## Channels

- Bugs: GitHub Issues (Bug Report)
- Feature requests: GitHub Issues (Feature Request)
- Questions: GitHub Issues with clear context
- Security: follow `SECURITY.md` (private reporting)

## Issue Quality

Please include:

- reproduction steps
- expected vs actual behavior
- OS / Flutter / Dart versions
- related logs (JSON/CLI output)

## Response Expectations

This is a community-maintained project. Immediate response is not guaranteed.
