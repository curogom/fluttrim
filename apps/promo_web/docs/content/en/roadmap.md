# Roadmap

Public roadmap for Fluttrim. Priorities may change based on product and maintenance needs.

## v1 Priorities

1. Harden core scan/plan/apply safety rules
2. Improve CLI usability and diagnostics
3. Improve desktop UX for project management and plan/apply flow
4. Improve macOS DerivedData attribution quality
5. Strengthen global cache safety rails
6. Improve FVM visibility and cleanup UX
7. Improve history persistence and comparison views
8. Continue docs/tests/CI quality work

## Operations

- Publish `fluttrim_core` first on pub.dev
- Keep Cloudflare Pages docs/website up to date
